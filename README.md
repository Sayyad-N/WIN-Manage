## WIN-Manage Tool (English)
- it's  simple tool for manage windows and based on python

## Functions
- Management Users
- Apply Extreme protection for Enterprise
- Get System Info

## Note
- In Extreme protection use admin for run it (script will auto admin run <3)

## Reqirements
- Python3
- install wmi
  ``` pip install wmi ```
